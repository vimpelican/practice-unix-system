Overview

In this chapter, I learned about overview of UNIX. For example - What OS is, How to connect UNIX system using ssh, etc.

01 Overview of UNIX

  1. OS and UNIX
    • OS is a set of program that controls the computer.
  2. History of UNIX
    • First developed in 1969 at Bell lab by Ken Thompson and Dennis Ritchie
    • Re-written in the C language in 1973
  3. Unix variants
  4. Structure of UNIX
    • Kernel, shell, utility
    • Kernel : Manage all reseources of computers
    • Shell : Act as an intermediary - users and kernel. It interpretes commands from user and send them to kernel. It shows result on the monitor.
    • Utility : Offers a variety of utilities(e.g. programming tool, network tool)

02 How to connect UNIX system and CLI

  1. Connect to UNIX system
    1. Open terminal(in Mac)
    2. Enter "ssh -oHostKeyAlgorithms=+ssh-dss -p [PORT] [ID]@comunix.seoultech.ac.kr"
    3. Log-in with given ID and PW
  2. How to edit command line
  3. Structure of command
    • commands [options] [arguments]
  4. How to use commands
    • Only command : ls
    • Command with option : ls -a
    • Command with argument : ls /tmp
    • Command with both option and argument : ls -a /tmp
  5. Basic usage of command
    • banner - print large banner on screen
    • date - display or set date and time
    • clear - clear the terminal screen
    • man - display online manual documentation pages
    • passwd [args] - modify a user's password
  6. Disconnect to UNIX system
    • logout : Only available on login shell
    • exit
    • [control]+[D]
