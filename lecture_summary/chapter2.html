Overview

In this chapter, I learned what directory is. There are some commands - pwd, cd, ls, mkdir, rmdir.

01 UNIX file system

  1. Types of file
    • Default file(text or binary)
    • Directory file : Its content is name of another file or sub-directory
    • Symbolic link file : Shortcut in Windows
    • Device file(block device file or character device file) : In /dev or /devices
  2. Directory structure(tree)
    • e.g.) /bin, /etc/default/login, /usr/bin, /usr/lib, /tmp, /etc
  3. Absolute path, relative path
    • If user is in /home/user1/Unix:
    • Absolute path(starts from root) : /home/user1/Unix
    • Relative path : Unix
  4. Rules of naming files and directories
    • Use alphabet, number, -, _, .
    • Can't use blank, *, &, |, ", ', ãƒ», ~, #, $, (, ), \, :, <, >.
    • Be case sensitive(aaa!=AAA).
    • A file starts with .(dot) - It is hidden file.

02 Check present working directory(in absolute) - pwd

03 Change directory - cd

  • cd : change to home directory
  • cd ~ == cd
  • cd [Name of directory]
  • cd .. : change to parent directory

04 Check file list of directory - ls

  1. Check file list of current directory
  2. Check file list of specific directory
  3. Check file information
  4. View detailed information - [-l] option
  5. Check information of directory itself - [-d] option

05 Make directory - mkdir

  1. Make one directory
  2. Make several directories at same time
  3. Automatic making of middle directory

06 Remove directory - rmdir
