Overview

In this chapter, I learned how to manage files. There are some commands - cat, more, detail, cp, rm, mv, ln, touch

01 View file contents : cat/more/tail

  1. Concatenate and print files : cat
  2. Print on a screen basis : more
  3. Display the last part of a file : tail

02 Copy files : cp

  1. Copy a file to file
  2. Copy a file to directory
  3. Copy several files to directory
  4. Using [-i] option
  5. Copy directories

03 Remove files(directory entries) : rm

  1. Basic file removal
  2. File removal using [-i] option
  3. Remove directories

04 Move files : mv

  1. Move from a file to file
  2. Move from a file to directory
  3. Move several files to directory
  4. Move using [-i] option
  5. Move from directory to directory

05 Link files : ln

  1. Hard link
  2. Symbolic link

06 Change file access and modification times : touch

  1. Create a file
  2. Change file access and modification times
